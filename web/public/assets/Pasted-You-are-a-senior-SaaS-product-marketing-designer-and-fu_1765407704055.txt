You are a senior SaaS product marketing designer and full-stack web developer.

Design and generate a complete, production-ready, responsive marketing website for a B2B SaaS product named “EngageNinja”.

EngageNinja is an AI-first, WhatsApp-first customer engagement platform for agencies and WhatsApp-heavy SMBs. It competes with Brevo.com and Interakt.com, but differentiates with:
- WhatsApp-first UX (not an email retrofit)
- Built-in AI campaign generation
- One-click resend to non-readers
- Built-in uplift & ROI proof
- Agency-native multi-tenant operations

The site’s goal is **conversion** (free trial / book demo), **trust**, and **clear differentiation**.

---

### 🧱 TECH STACK & OUTPUT FORMAT

- Use **Next.js + React + Tailwind CSS**
- Use **component-based structure**
- Provide **fully working page layouts**
- Include **mock data and placeholder images/illustrations**
- SEO optimized
- Accessibility compliant (WCAG AA)
- Output complete files where possible: `app/layout.tsx`, `app/page.tsx`, `app/(marketing)/*`, `components/*`
- Use clean professional SaaS UI styling similar to Brevo + Linear + Stripe

---

### 🎨 VISUAL STYLE & THEMING

Overall style:
- **Modern SaaS with glassmorphism & “liquid” gradients**
- Soft, blurred panels on a gradient background
- Subtle frosted-glass cards with borders and shadows
- Plenty of icons and simple illustrations to support copy (can be SVG placeholders)

Light/Dark theme:
- Implement **full light + dark theme support** using Tailwind and CSS variables (or `next-themes` style approach).
- Add a **theme toggle** (sun/moon icon) in the header.
- Make sure all sections, cards, buttons, and text look good in both themes.
- Persist theme preference in local storage.

Color direction:
- Trustful teal/green/blue primary
- Highlight states for success, warning, error
- Ensure contrast ratios are accessible in both light and dark modes.

---

### 📱 RESPONSIVE & INTERACTION

- **Mobile-first, fully responsive** (mobile, tablet, desktop)
- Navigation collapses into a clean mobile menu with the main CTAs easily visible.
- Sections stack gracefully on small screens.
- Keep tap targets large enough on mobile.
- Use simple, tasteful animations:
  - Subtle hover effects on cards/buttons
  - Gentle movement of background “liquid” shapes
  - Animated number counters for metrics
